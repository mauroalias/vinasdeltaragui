# 🍷 Viñas del Taragüí

Sitio web de vinoteca desarrollado con Laravel.

## 📌 Descripción

Proyecto web que permite visualizar productos como vinos y bebidas, con navegación, contacto y página de términos y condiciones.

## ⚙️ Tecnologías

- Laravel
- PHP
- Bootstrap

## 🚀 Instalación

1. Clonar repositorio:
git clone https://github.com/mauroalias/vinasdeltaragui.git

2. Instalar dependencias:
composer install

3. Copiar archivo de entorno:
cp .env.example .env

4. Generar clave:
php artisan key:generate

5. Levantar servidor:
php artisan serve

## 📂 Estructura

- `resources/views/frontend` → vistas del usuario
- `routes/web.php` → rutas
- `components` → layout, navbar, footer
